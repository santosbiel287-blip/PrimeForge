PrimeForge Engine 🚀
Bem-vindo ao PrimeForge Engine, um estúdio de desenvolvimento no-code focado em agilizar a criação de vitrines e lojas digitais.

⚠️ Estado do Projeto: Beta
O PrimeForge Engine está atualmente em fase de desenvolvimento ativo. Esta versão é uma demonstração da arquitetura e das funcionalidades principais. Não é possível lançar ou publicar sites operacionais com este sistema no momento.

Como Funciona
O PrimeForge Engine foi projetado para atuar como um painel de controle (Engine) onde o usuário centraliza a gestão de seus projetos:

Editor Visual: Interface intuitiva onde você edita informações do produto (nome, preço, descrição) e visualiza o resultado em tempo real.

Gestão de Vitrine: Sistema que processa a entrada de dados do editor e gera uma visualização pronta para o cliente final.

Hierarquia de Planos: Sistema em desenvolvimento para segmentar o acesso, onde usuários gratuitos têm limitações de recursos (como número de sites), enquanto planos avançados desbloqueiam domínios próprios e escalabilidade.

Objetivo do Beta
O foco atual é estruturar a base técnica (o motor do software) para garantir que:

A experiência do usuário no editor seja fluida.

A lógica de permissões de planos seja robusta e segura.

A integração visual entre o painel de edição e o preview da loja funcione de forma coerente.

Como contribuir
Como este é um projeto de aprendizado e desenvolvimento de Software Engineering, sugestões e críticas construtivas à estrutura do código são bem-vindas. Sinta-se à vontade para explorar os arquivos no diretório frontend para entender como o esqueleto do sistema foi montado.
